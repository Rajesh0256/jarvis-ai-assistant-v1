## üéØ Command Tips

### For Best Results:

1. **Be Clear and Specific:**
   - ‚úÖ "open file report.pdf"
   - ‚ùå "open that file"

2. **Include File Extensions:**
   - ‚úÖ "find file document.txt"
   - ‚ö†Ô∏è "find file document" (works but less accurate)

3. **Use Full Location Names:**
   - ‚úÖ "weather in New York"
   - ‚ùå "weather in NY" (might not work)

4. **Speak Naturally:**
   - ‚úÖ "Jarvis, what's the weather in Paris?"
   - ‚úÖ "Jarvis, tell me the weather in Paris"
   - Both work!

5. **Wait for Response:**
   - Give Jarvis time to process each command
   - Wait for confirmation before next command

## üìù Command Patterns

### Pattern 1: Open + Name
```
"open [website/app name]"
Examples:
- "open Instagram"
- "open Chrome"
- "open Calculator"
```

### Pattern 2: Open File/Folder + Name
```
"open file [filename]"
"open folder [foldername]"
Examples:
- "open file report.pdf"
- "open folder documents"
```

### Pattern 3: Action + File/Folder + Name
```
"[action] file [filename]"
"[action] folder [foldername]"
Examples:
- "delete file old.txt"
- "create folder projects"
- "find file report.pdf"
```

### Pattern 4: Weather + Location
```
"weather in [location]"
"what's the weather in [location]"
Examples:
- "weather in Paris"
- "what's the weather in Tokyo"
```

### Pattern 5: Info Request
```
"show info for [filename]"
"what's the size of [filename]"
Examples:
- "show info for report.pdf"
- "what's the size of document.txt"
```

## üö´ Common Mistakes

### ‚ùå Don't Say:
- "open the file called report" (too vague)
- "delete that folder" (not specific)
- "find my document" (which document?)
- "weather" (which location?)

### ‚úÖ Instead Say:
- "open file report.pdf"
- "delete folder temp"
- "find file document.txt"
- "weather in New York"

## üé§ Voice Recognition Tips

1. **Speak Clearly:** Enunciate each word
2. **Normal Pace:** Not too fast, not too slow
3. **Quiet Environment:** Reduce background noise
4. **Good Microphone:** Use a quality microphone
5. **Consistent Volume:** Maintain steady voice level

---

## üîß If Commands Don't Work

1. **Check the command format** (see examples above)
2. **Verify file/folder exists** in common locations
3. **Try with file extension** (e.g., .pdf, .txt)
4. **Check internet connection** (for weather, websites)
5. **See TROUBLESHOOTING.md** for detailed help
